# Subway Surfers

## Description

This game is a replica of Subway Surfers and developed using only WebGl. The code is completely modular and open to contributions. Works on all browsers.

## Features and Bonus

* Jetpack
* Magnet
* Jumping boots
* Coins
* Textured graphics
* Grayscale graphics
* Flashing walls

## To Run

    ```console
	bar@foo:~$ firefox index.html
	```
